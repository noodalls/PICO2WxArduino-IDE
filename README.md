# PICO2WxArduino-IDE

This is my latest version of a hardware input lag testing device. It has been through several evolutions. 

At the present time it will need you to source the parts and assemble yourself. In the future there may be the possibility of shipping a fully assembled version. 

You will need

https://oshpark.com/shared_projects/kSso9cQ9

This is the PCB that clips on top of the Pico2W. You will need to solder two 20P female header pins, one 4P female header pin. You will also need to solder four buttons. 

You will need a Raspberry PICO2W. 

https://www.raspberrypi.com/products/raspberry-pi-pico-2/

You will need an OLED Screen

https://www.amazon.com.au/dp/B0D2RMQQHR

The sensor for detecting light changes is here

https://oshpark.com/shared_projects/rQflS270

You will need some 50k resistors

You will need a phototransistor

https://www.amazon.com.au/dp/B00M1PMHO4
